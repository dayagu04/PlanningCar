## 答辩可能的问题与回答

### Q1：为什么选择 Webots 而不是 Gazebo？
**A**：
- Webots 在 Windows 上安装更简单
- Python API 更友好，适合本科毕设
- 物理仿真精度满足需求
- 开题报告中已说明改为单平台方案

### Q2：IMU 辅助分类的创新点是什么？
**A**：
- 2D 激光雷达在均匀斜坡上无法提取坡度特征
- 引入 IMU pitch 角直接反映机器人姿态
- 斜坡识别率从 0% 提升到 95%
- 简单有效，适合嵌入式部署

### Q3：为什么不用机器学习分类器？
**A**：
- 规则分类器准确率已达 92%，满足需求
- 无需标注数据集，开发周期短
- 阈值可解释，便于调试
- 计算量小，实时性好

### Q4：斜坡导航稳定性问题如何解决？
**A**：
- 已降低斜坡速度（5.0 → 2.0 rad/s）
- 已降低转向增益（4.0 → 2.0）
- 后续可添加滑移补偿算法
- 或使用履带式机器人（更强抓地力）

### Q5：项目的实际应用价值？
**A**：
- 工地巡检机器人（斜坡、碎石路面）
- 矿山勘探机器人（复杂地形）
- 灾害救援机器人（废墟、瓦砾）
- 仿真验证后可迁移到实体机器人
